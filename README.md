# mbot, mice bot I guess
This is a discord bot for personal usage- it will accrue features
as I want them.

# Building
```sh
. .pub_env
cargo build
```

# Running
Set the environment variable `MBOT_TOKEN` to the bot token you're using, then run `mbot`

That's it.
